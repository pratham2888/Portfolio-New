# Pratham Bhanushali — Portfolio

Personal portfolio website with an integrated rule-based chatbot.

## Files

- `index.html` — Main HTML structure
- `style.css` — All styles (dark editorial theme)
- `script.js` — Cursor, animations, mobile nav, chatbot logic

## Features

- Dark editorial design with amber accent
- Custom cursor + follower animation
- Scroll reveal animations
- Sticky responsive nav with mobile hamburger
- AI-style chatbot (fully rule-based, no API, no dependencies)
- Sections: Hero, About, Experience, Projects, Research, Contact

## Chatbot

The chatbot is 100% static — no API key, no backend, no external calls.
It uses keyword pattern matching against a hardcoded knowledge base about Pratham.
Topics covered: skills, experience, projects, research, education, contact, job search.
